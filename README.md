## 👋 歡迎來到境流社群

這裡是 **Ambientype (境流)** 的官方社群中心。

Ambientype 是一款專為 Windows 設計的次世代輸入法，由 **Rust** 語言打造核心，追求極致效能與記憶體安全。我們利用 **Local AI (本地運算)** 提供精準預測，並導入 **Japandi (日式北歐)** 極簡美學，讓輸入法如空氣般自然，不打斷您的心流體驗。

> **注意：** Ambientype 為閉源專案 (Closed Source)。本倉庫 (Repository) 專用於社群互動、問題追蹤與版本發布。

## ⚡ 核心特色

* **Rust 鑄造 (Built with Rust)**：
    拒絕卡頓。擁有極低的記憶體佔用與毫秒級的響應速度，確保系統穩定不崩潰。
* **智慧熔斷 (Smart Fuse)**：
    專為玩家設計。自動偵測遊戲情境與重複按鍵（如 WASD），智慧防誤觸，絕不搶奪視窗焦點。
* **本地隱私 (Local Intelligence)**：
    您的數據屬於您。所有的 AI 預測運算皆在本地完成，絕不將按鍵紀錄上傳雲端。
* **極簡美學 (Japandi Aesthetics)**：
    引入 Acrylic 毛玻璃特效與動態光暈，打造與 Windows 11 完美融合的視覺體驗。
